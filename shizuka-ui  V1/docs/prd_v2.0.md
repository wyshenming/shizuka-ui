# 项目需求文档 (PRD) - 静谧之声 (Shizuka UI)

## 1. 项目愿景
“静谧之声”是一款专为 Galgame 爱好者和角色扮演用户设计的沉浸式聊天应用。它通过极简的视觉语言、大量的留白和低饱和度的色彩，消除传统社交软件的压迫感，为用户提供一个安静、轻松、充满文学气息的创作与交流空间。

## 2. 核心设计原则
*   **静谧感 (Tranquility)**：采用低饱和度色调（如 #a5cad2），减少视觉噪音。
*   **呼吸感 (Breathability)**：强调留白（Negative Space），布局通透，不堆砌功能。
*   **沉浸感 (Immersion)**：弱化工具属性，强化叙事感，如使用衬线体（Noto Serif）和诗意的文案。
*   **极简交互 (Minimalist Interaction)**：操作逻辑直观，界面纯净，关键功能在需要时才优雅浮现。

## 3. 功能模块说明

### 3.1 角色画廊 (Character Gallery)
*   **展示形式**：采用纵向长方形列表，一行一个角色卡片。
*   **核心功能**：
    *   **极简搜索**：顶部胶囊型极简搜索框，支持快速检索角色。
    *   **角色管理**：卡片右上角设有删除与修改按钮，方便维护。
    *   **角色导入**：点击导入按钮弹出毛玻璃效果菜单，支持“导入角色卡”或“新建角色”。
*   **跳转逻辑**：点击“新建角色”跳转至编辑页。

### 3.2 角色对话 (Roleplay Chat)
*   **风格定位**：极简气泡版。
*   **视觉规范**：
    *   圆润的消息气泡，支持双方头像显示。
    *   顶部展示角色名字，左侧返回，右侧设有“角色详情”按钮。
    *   底部输入框与发送按钮垂直居中对齐，维持完美的视觉平衡。
*   **交互体验**：较大的垂直间距，营造手账般的翻阅感。

### 3.3 回忆回顾 (Memory Review)
*   **风格定位**：纯文字叙事美学。
*   **视觉规范**：
    *   移除所有图片干扰，回归文字本源。
    *   顶部嵌入文学性对白（如：“那些被光影镌刻的无声对白”），采用轻盈斜体。
    *   沉浸式设计：移除底部导航栏，增加左上角返回按钮。

### 3.4 角色编辑 & 个人资料 (Editor & Profile)
*   **核心功能**：
    *   **基础设定**：姓名、头像、性格描述、年龄、性别等。
    *   **动态扩展**：支持点击“添加自定义字段”实时增加空白文本框，且每个字段均可删除。
    *   **保存机制**：右上角醒目的保存按钮，确保设定即时生效。

### 3.5 系统设置 (System Settings)
*   **API 设置**：包含 OpenAI URL、API Key（脱敏显示）、模型选择及测试连接功能。
*   **关于页面**：极致留白设计，展示项目寄语及版本号、GitHub 入口。

## 4. 技术与视觉规范
*   **设计系统**：Shizuka UI
*   **字体**：NOTO_SERIF (衬线体)
*   **主色调**：#a5cad2 (灰蓝色系)
*   **组件特征**：ROUND_EIGHT (8px 圆角)；毛玻璃效果 (Backdrop Blur) 用于弹出层。
*   **适配设备**：Mobile (移动端优先)

## 5. 后续规划
*   支持更多 Galgame 风格的转场动画。
*   增加角色语音包导入与预览功能。
*   优化长文本在回忆页面的排版渲染。
